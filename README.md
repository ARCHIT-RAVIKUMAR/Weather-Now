# 🌤️ Weather Now

A simple weather app to get real-time weather for any city in the world.

## Features
- 🔍 Search weather by city name
- 📍 Auto-detect your location
- 🌡️ Toggle between °C and °F
- 💧 Humidity, wind speed & feels like
- 📅 5-day forecast

## Built With
- HTML, CSS, JavaScript
- [OpenWeatherMap API](https://openweathermap.org/api)
